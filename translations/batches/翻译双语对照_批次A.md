### 单元 G01-U06｜老师｜回信

> 原编号：1-7, 1-8

#### P01

原文：

```md
詩意を書いているんですね。
早速拝見いたしました。詩の節々に音符と和音のコードも付いていますね。私は、音楽に関しては門外漢というわけでもなく、多少の譜読みなら行うことは出来ます。
楽譜以外の場所に音符が書いてあるのは少し不思議な惑ドがしますね。何か物を飾りつけるように、文字に音がついているような想像が浮かびました。
```

译文：

```md
你是在写带着诗意的东西啊。
我已经立刻读过了。诗句的各处还标了音符和和弦代码。说到音乐，我倒也不算全然是门外汉，稍微看看谱子还是做得到的。
音符被写在乐谱之外的地方，会让人觉得有一点奇异。像是在给什么东西做装饰似的，我不由得想到，那些字仿佛也带上了声音。
```

#### P02

原文：

```md
「花も騒めく」について
夏が来たことを喜ぶ詩でしょうか。
白妙という言葉が何度も出てきますね。続けて雲や袖という単語を使っているので、秋詞としての白妙を意識しているのでしょうね。
白妙に、照れるべき月も隠れ、と並ぶと万葉集を連想させられます。

まそ鏡照るべき月を白たへの雲が隠せる天つ霧かキ

たったかな。今は手元に資料がないので正確かわかりませんが。音には栲という、木の樹皮から作った布がありました。白栲はつまり白い布で、それを用いて作られた衣を指す言葉でもあり
```

译文：

```md
关于《花也骚动》。
这是一首为夏天到来而欢喜的诗吗。
“白妙”这个词出现了很多次。又接着用了“云”和“袖”这样的词，所以你大概是有意识地在用它作为秋词的那一层意义吧。
“白妙”和“连本该发亮的月也躲起来了”并列在一起，会让我联想到《万叶集》。

“まそ鏡照るべき月を白たへの雲が隠せる天つ霧かキ”

大概是这样吧。眼下我手边没有资料，所以不确定是不是准确。古时有所谓“栲”，是由树皮制成的布。白栲，也就是白色的布，同时也可以指用它做成的衣服。
```

#### P03

原文：

```md
ました。これははのまま白妙という色に転じます。
これを踏まえると、「照るべき月も隠れお前だけ光る白妙の袖」は、月が隠れて、夜の暗闇の中に白い袖が浮かんで見える様子を想像しました。こう考えると全編に亘って「お前」という言葉が指しているのは、白という色のことでもあるのかな。
```

译文：

```md
后来它又顺着这个意思，转成了“白妙”这种颜色本身。
这样一来，“本该发亮的月也躲起来了，只有你发亮的白妙衣袖”这一句，我就会想象成月亮隐去之后，白色的衣袖浮现在夜里的黑暗之中。这样想的话，贯穿全诗的那个“你”，说不定指的也是“白”这种颜色吧。
```

#### P04

原文：

```md
「雲になる」について
夏の畦道に雲が浮かんでいる様子が見えました。それと同時に何故か田舎の海も浮かびました。どうしてでしょうね。私の中で畦道は海の近い印象があるんでしょうね。故郷の印象が大きいのかも知れません。譜面は多少読めるので、これもメロディと一緒に口ずさんでみましたが、不思議な印象ですね。楽譜のように拍子ということになるのかな？その少しずれた不思議な感じが面白くて良いと思います。
```

译文：

```md
关于《变成云》。
我看见了夏天的田埂小路上浮着云的样子。与此同时，不知为什么，我也想起了乡下的海。为什么呢。大概在我心里，田埂总带着一种离海很近的印象吧。也许是故乡的印象太深了。我多少看得懂一些谱，所以也跟着旋律试着哼了哼，这首给人的感觉很奇异。是要按乐谱那样算拍子吗？那种微微错开的奇异感，我觉得很有意思，也很好。
```

```md

所以我在想，我诗里的表达，有没有什么奇怪的地方？或者说，这里如果改成这样会更自然，那里如果改成那样会更有技巧……从旁人的眼里看，有没有什么内容是应该修改的呢。我只是很在意这一点。
如果还能收到您的回信，我会很高兴。
请多关照。
敬具
```

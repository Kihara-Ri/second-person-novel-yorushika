#### P01

原文：

```md
拝啓
今日は手紙だけです。
最近どうにも筆が進まなくて、すみません。
だからよくテーマを探して散歩しています。
そういえば、先生から送られてくる手紙の右下に押してあるハンコは何のマークですか？花柄のそれです。末尾には名前を書くのが普通だと思っていましたが、サイン代わりの印鑑というのもあるんですね。手紙や詩の用紙の端に押して返してくれるのも実は結構気に入っています。
昨日、母が祖父と電話をしているようでした。夏が終わったあとに母の仕事が一段落するそうで、祖父の家に会いにいく段取りをつけていました。僕も行くかは悩んでいます。
祖父の家は少々田舎の方にあります。一方、僕が住んでいる場所は多少都会です。あ、送った封筒に住所が書いてあるからわかりますよね。
祖父のところに行くためには県をいくつか{跨|また}がなければいけないので、年に数度くらいしか顔を見せには行けません。母は東京に出て父と結婚したそうです。そのせいで僕自身は親戚にあったことはほとんどありません。顔を覚えている人はいないかも。でも母に弟がいることは知っています。先日開いたら、カメラとギターはその人から預かったものだそうです。
```

译文：

```md
敬启
今天只有这一封信。
最近实在怎么都写不出来，对不起。
所以我常常一边散步，一边找题目。
说起来，老师寄来的信右下角总会盖着一个印，那到底是什么图样？就是那个花样的印。我本来以为信末一般都是写名字，原来还有拿印章当签名的做法啊。其实，您把它盖在信纸或诗稿边上再寄回来这件事，我还挺喜欢的。
昨天，母亲好像和外祖父通了电话。她说等夏天过去，工作会稍微告一段落，到时候要安排去看外祖父。我还在犹豫自己要不要一起去。
外祖父住的地方有些偏乡下。相比之下，我现在住的地方多少算有点城市味。啊，不过看我寄去的信封地址，老师应该也知道吧。
要去外祖父那里，得跨过几个县，所以一年也只能去见上几次。听说母亲当年是来了东京以后才和父亲结婚的。正因为这样，我几乎没怎么见过亲戚。说不定谁的脸我都记不得了。不过我知道母亲有个弟弟。前些天才知道，相机和吉他原来都是那个人寄放在这里的。
```

#### P02

原文：

```md
祖父は広い家に一人で住んでいます。祖母はいません。数年前大きな怪我をきっかけに入院生活を送って、そのまま亡くなりました。
それ以来、祖父はずっと寂しそうに見えます。
初めて先生に手紙を送った時に、「雲になる」という詩を入れましたよね。先生が、田舎の海を想像するといったあれです。
あれは祖父のいる田舎で見た景色です。母の書斎にあった雲に関する本を読んだ時に、それが浮かびました。祖父の家から出てすぐに畦道があります。僕は祖母とよくそこを歩きました。畦道の向こう側には雲が浮かんでいます。滑らかな白の入道雲です。雲になってみたいと僕はよく祖母に言いました。青い海の中をとても気持ちよさそうに泳いでいるように見えたからだと思います。
今でもよく、そう考えます。
何となく思い出したので書きました。雲の写真を入れておきます。実を言うと、この前母のカメラを持って散歩に出たら、あまりも綺麗な雲が撮れたので、本当の目的はこれを送ることです。
前段はその言い訳ですね。
今週、また図書館に行ってみようと思います。
お身体にお気をつけお過しください。
敬具
```

译文：

```md
外祖父一个人住在很大的房子里。外祖母已经不在了。几年前，她因为一次很重的伤住进了医院，后来就那样去世了。
从那以后，外祖父看上去一直都很寂寞。
我第一次给老师写信的时候，放进去过一首《变成云》，就是老师说会让人想到乡下海边的那首。
那其实就是我在外祖父那边的乡下见过的景色。是在读母亲书房里那本和云有关的书时，我忽然想起来的。外祖父家一出门就是田埂小路。我以前常和外祖母在那里走。田埂那边浮着云，是那种光滑雪白的入道云。我总对外祖母说，真想变成云。大概是因为，它们看起来像在蓝色的海里游得很舒服。
到现在，我也还是常常这样想。
只是忽然想起来了，所以写给您。信里我放了云的照片。老实说，前些天我拿着母亲的相机出去散步，拍到了一张漂亮得过分的云，所以真正想寄来的其实是这个。
前面那些话，不过是它的借口罢了。
这周我还想再去一趟图书馆。
请您保重身体。
敬具
```

> 原编号：9-1, 9-2

#### P01

原文：

```md
拝啓
続けて二通送ってしまって、すみません。
荘子を読み終わりました。合わせて、孔子も読みました。
島崎藤村を何冊か借りました。朔太郎の他の詩集を借りました。平安時代の和歌が気になったので、{古今|ここん}和歌集と詞花和歌集を借りました。
図書館は凄いですね。こうして先生にきっかけをもらうまで、僕は学校以外の図書館に行ったことはありませんでしたから、新鮮です。母の利用者カードがリビングに置かれていたので、それを{拝借|はいしゃく}して、ここ最近は入り{浸|ひた}ってます。
他人の言葉って、なんて良いんでしょうか。僕は自分の言葉には胸は動いたことはありませんが、他人が書いたというだけでこれだけ別の物に感じられるなんて。
これだけあれば、また新しい何かが出てくる気がします。そして、先日の「太陽」のときは、テーマをいただけて、本当に助かりました。何だか嬉しかったんです。よかったら、また何かくださいませんか？それで、他にも読んでおいた方がいい本があれば、教えていただけたら嬉しいです。

僕はこれから図書館に籠ることになると思います。
次に送る手紙は、少し日が開くと思います。しばらく、時間をください。
敬具
```

译文：

```md
敬启
一连寄了两封信，对不起。
我已经把《庄子》读完了。顺便也读了《孔子》。
我借了几本岛崎藤村，也借了朔太郎其他的诗集。因为开始在意平安时代的和歌，所以又借了《古今和歌集》和《词花和歌集》。
图书馆真厉害啊。在老师给我这个契机之前，我从没去过学校以外的图书馆，所以一切都很新鲜。母亲的借书证刚好放在客厅里，我就借来用了，这阵子几乎整个人都泡在里面。
别人的话，怎么会这么好呢。我从来没有被自己的词句真正打动过，可只因为是别人写下的，它们竟然就能变得这样不一样。
有了这些东西，我总觉得还能再长出什么新的东西来。还有，上次那个“太阳”的题目，老师给得真的帮了我很大的忙。我其实很高兴。如果可以的话，您还能再给我一点什么吗？另外，如果还有什么值得先读的书，也请告诉我，我会很高兴。

接下来我大概会一直待在图书馆里。
下次寄信，应该会隔得稍微久一点。请先给我一点时间。
敬具
```

```md
拝啓
家の近所に、びわの実が{成}^[生]りました。先生は以下がお過しでしょうか。
びわは火曜日の昼に図書館から帰る途中で見かけました。鮮やかな{橙|だいだい}色の、丸い実です。{生垣|いけがき}越しで近くには寄れませんでしたが、線の葉のなかに橙色が見え隠れして、目を{惹|ひ}きます。その時僕は丁度、北原白秋の{「花の桐」}^[「桐の花」の間違い？]を読んでいて、目を落とすと示し合わせたようにページ上のびわの文字が目に入りました。
それはこういう短歌でした。

琵琶の木に黄なる琵琶の実がかがやくとわれ驚きて飛びくつがへる

「飛びくつがへる」は飛んでひっくり返ることだそうです。僕はそんな風に飛び上がったりはしませんでしたが、目の前にあるびわの実と、ページの角ばった文字を見比べながら、何だか嬉しいような、偶然文字と世界と、僕の手のひらが浅く繋がったような、そういう感覚を覚えました。平たく言ったら、運命だと思ったのです。それで、今日はびわをモチーフに詩を書いてみようと思いました。
```

译文：

```md
敬启
我家附近的枇杷结果了。老师近来过得如何。
那枇杷是星期二中午从图书馆回来时看见的。是鲜亮的{橙|だいだい}色，圆圆的果子。隔着树篱，我没法走近，可那一点点橙色在长叶间若隐若现，很抓人眼睛。那时候我正好在读北原白秋的《桐之花》，低头一看，像是约好了一样，书页上刚好也落进了“枇杷”这两个字。
那是一首这样的短歌。

“琵琶树上黄了的枇杷果正闪闪发亮，我一惊，几乎要整个人翻过去。”

听说“飞びくつがへる”就是跳起来翻过去的意思。我倒没有真那样跳起来，只是看着眼前的枇杷果，再看看纸页上方方正正的文字，忽然生出一种说不出的高兴，像文字、世界和我的手掌只浅浅地接到了一下。说得直白一点，我觉得那像命运。
所以今天，我想试着写一首以枇杷为题的诗。
```

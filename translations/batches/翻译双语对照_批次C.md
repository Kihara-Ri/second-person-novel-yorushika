> 原编号：16-1.1

#### P01

原文：

```md
拝啓
湿気が増えて、入道雲が高く昇り始めています。
何だかいつもと違う風に歌詞が書けました。これはなるべく楽しそうになるよう意識した詩です。僕にしては悲観的過ぎなくていいのかもしれません。
この詩ははっきりとしたイメージがあります。踊っている女性です。ルバートは音楽の言葉で、テンポを揺らしながら自由に演奏することです。
> 知りませんでした。ジャズみたいな感じかな？
> その通りです。先生も知らない言葉があるんですね。
メロディもテンポもイメージがあります。そういう時は詩よりも、もっと歌詞っぽくなりますね。
一度、見てみてください。先生の印象を聞かせてください。
敬具
```

译文：

```md
敬启
湿气渐渐重了，入道云也开始越升越高。
这次我写出来的词，和以前有点不一样。这是一首我刻意想让它显得尽量快乐些的诗。像我这样的人，也许偶尔不那么悲观一点，反而会更好。
这首诗在我脑子里有很清楚的画面。是一个正在跳舞的女人。Rubato 是音乐里的词，意思是一边摇动节拍，一边自由地演奏。
> 这个我还真不知道。是不是有点像爵士那样？
> 对，就是那种感觉。原来老师也会有不知道的词啊。
旋律也好，节拍也好，我心里都有很具体的形象。这种时候，写出来的东西就会比“诗”更像歌词一些。
请您看一看吧。我想听听老师的印象。
敬具
```

### 单元 G16-U02｜老师｜回信

> 原编号：16-1.2

#### P01

原文：

```md
君の詩はいつもそういう傾向がありそういう傾向がありますよ。頭の中にあるメロディが感じられて、私はそこを好ましく思います。そして、懐かしさを感じました。フィルム映像みたいに色が褪せているような、そういう懐かしさです。
今回の詩はとても女性的ですね。一人称が私で、語尾の表現が「わ」だからでしょうね。どうしていつもは変えようと思ったんですか？何をイメージしたのか、言葉にしてみてください。
```

译文：

```md
你的诗本来就一直有这种倾向。能让人感觉到脑子里那条旋律，而我一直很喜欢这一点。还有，这次我读出了某种怀旧感。像胶片影像褪了色那样的，那种怀旧。
这首诗非常女性化。大概是因为第一人称用了“私”，句末又用了“わ”吧。你为什么会想和往常不一样地去写呢？把你脑子里想象的东西，说出来试试看。
```

```md
この間から返信用に便箋を入れてくれていますよね。ありがとうございます。気を遣わせてしまったようで、すみません。

それは、女性の姿に母を重ねてイメージしたからです。
```

译文：

```md
8/18

从前些天开始，老师您都特地放了信笺当回信用吧。谢谢您。让您这样费心，真对不起。

那首诗之所以写成那样，是因为我把母亲叠在了那个女人的形象上去想。
老师以前说过，让我试着写写我自己吧。顺着那句话，我脑子里浮起来了一段画面。
是一个女人，在空无一物的空间里跳舞。看不见脸。可我就是隐约知道，那是母亲。
如果直接把她当题材，我会嫌它离自己太近，所以我只拿了那个意象。也正因为这样，我写出来的样子，也许会比实际更快乐一些。我的母亲一直都带着一张不幸福的脸。她偶尔也会借外祖父的手帮忙，可基本还是独自一人把我养大。早上在我起床之前就出门，晚上很晚才回来。她大概并不是真的穷。毕竟她有书房，也从没跟我提过钱上有什么困窘。也许，那也只是她在逞强罢了。
```

```md
昨天，我拿着母亲的烟和火柴出了门。撕开包装的时候，一股像香辛料似的味道扑了出来。
```
